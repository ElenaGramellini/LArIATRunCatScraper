# LArIATRunCatScraper

Description of this Package:
This python script retrieves the run conditions from the LArIAT run catalogue, given a csv containing
run, subrun, event number. 

What does this package contain?

<ol>
  <li>README.md</li>
  <li>ScanPage.py</li>      
  <li>test.csv</li>
</ol>
